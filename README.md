# Modula Docs

Documentation and guides for Modula.

## Purpose

This module provides a dedicated docs surface inside Modula.

It is intended to expose:
- getting started guides
- module and plugin documentation
- Board and Marketplace help
- developer workflow references

## Surfaces

- Sidebar page: `/docs`
- Standalone page: `/docs`

## Source Of Truth

Docs is maintained in GitHub as the editable source of truth.

Deployment flow:

- GitHub repo
- GitHub release artifact
- Marketplace submission/update
- Modula runtime refresh
- live surfaces update
